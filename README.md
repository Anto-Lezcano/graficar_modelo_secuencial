# 📊 Regresión Lineal Simple con TensorFlow y Flask

Este proyecto es una aplicación web interactiva que implementa un modelo de regresión lineal simple utilizando TensorFlow y Flask. Permite entrenar el modelo con datos personalizados, visualizar la línea de regresión y realizar predicciones múltiples de forma sencilla.

---

## ✨ Características

- Backend en Flask que gestiona el entrenamiento y predicción
- Modelo de regresión lineal simple con TensorFlow
- Soporte para múltiples valores de entrada en la predicción
- Visualización de la pérdida durante el entrenamiento

---

## ✅ Requisitos

Instala las dependencias necesarias con:

````bash
pip install flask tensorflow numpy

---

Ejecuta el servidor:
````
